# Contract: Release Pipeline (`release.yml`)

**Workflow file**: `.github/workflows/release.yml`
**Feature**: Open Source Infrastructure & CI/CD | [plan.md](../plan.md)

---

## Trigger Contract

| Event | Pattern | Who Can Trigger |
|-------|---------|----------------|
| `push` (tag) | `v[0-9]+.[0-9]+.[0-9]+` | Repository maintainers with push access only |

**Non-matching tags** (e.g., `v1.0.0-beta`, `test-tag`, `latest`): Pipeline does NOT trigger.
**Fork contributors** cannot push tags to the upstream repository; they cannot trigger this workflow.

---

## Job: `release`

### Required Secrets

| Secret | Purpose | If Missing |
|--------|---------|-----------|
| `CERTIFICATE_P12_BASE64` | Developer ID Application certificate (.p12) | Job fails at sign step |
| `CERTIFICATE_P12_PASSWORD` | Passphrase for the .p12 file | Job fails at sign step |
| `KEYCHAIN_PASSWORD` | Temporary keychain password | Job fails at sign step |
| `NOTARIZATION_APPLE_ID` | Apple ID for notarytool | Job fails at notarize step |
| `NOTARIZATION_TEAM_ID` | Apple Developer Team ID | Job fails at notarize step |
| `NOTARIZATION_APP_SPECIFIC_PASSWORD` | App-Specific Password for notarytool | Job fails at notarize step |

### Steps (ordered, each must succeed before the next)

| # | Step | Action | Output |
|---|------|--------|--------|
| 1 | Checkout | `actions/checkout@v4` with `fetch-depth: 0` (full history for changelog) | Repo at tag |
| 2 | Select Xcode | `sudo xcode-select -s /Applications/Xcode_16.2.app` | Pinned toolchain |
| 3 | Resolve packages | `xcodebuild -resolvePackageDependencies` | SPM cache populated |
| 4 | Build Release | `xcodebuild build -scheme Drawl -configuration Release` | `build/Build/Products/Release/Drawl.app` |
| 5 | Run tests | `xcodebuild test -scheme DrawlTests` | Must pass; blocks release if tests fail |
| 6 | Import certificate | Decode `CERTIFICATE_P12_BASE64` → import to temp keychain | `build.keychain` ready |
| 7 | Sign app | `codesign --deep --force --options runtime --sign "Developer ID Application: ..."` | Signed `.app` |
| 8 | Package DMG | `bash scripts/create_dmg.sh --app-path build/Build/Products/Release/Drawl.app` (skip xcodebuild; package already-signed app) | `build/Drawl.dmg` |
| 9 | Notarize | `xcrun notarytool submit build/Drawl.dmg --wait --timeout 30m` | Notarization ticket from Apple |
| 10 | Staple | `xcrun stapler staple build/Drawl.dmg` | Ticket embedded in DMG |
| 11 | Validate | `xcrun stapler validate build/Drawl.dmg` | Confirms staple succeeded |
| 12 | Clean keychain | `security delete-keychain build.keychain` | Temp keychain removed |
| 13 | Create Release | `gh release create $TAG --generate-notes --title "Drawl $TAG" build/Drawl.dmg` | GitHub Release published |

### Outputs

| Output | Type | Description |
|--------|------|-------------|
| GitHub Release | GitHub entity | Tagged release with changelog |
| `Drawl.dmg` | Release asset | Signed, notarized, stapled DMG |

### Failure Behaviour

| Step Fails | Consequence |
|-----------|-------------|
| Steps 1–5 (build/test) | Job fails; no signing secrets used; no GitHub Release created |
| Steps 6–12 (sign/notarize) | Job fails; keychain cleanup step runs regardless (`if: always()`); no GitHub Release created |
| Step 13 (publish) | Job fails; maintainer must manually create release or re-run the job |

**Invariant**: A GitHub Release is only published if steps 1–12 all succeed. No partial
releases are published. A failed run leaves no public artifacts.

## Artifact Contract

### DMG File

| Attribute | Value |
|-----------|-------|
| Filename | `Drawl.dmg` |
| Contents | Signed `Drawl.app` + `/Applications` symlink |
| App signature | `Developer ID Application` identity, hardened runtime |
| Notarization | Submitted to Apple, ticket stapled to DMG |
| Gatekeeper check | `spctl --assess --type open --context context:primary-signature "Drawl.app"` → `accepted` |
| Staple check | `xcrun stapler validate Drawl.dmg` → `The staple and validate action worked!` |

### GitHub Release

| Attribute | Value |
|-----------|-------|
| Tag | `v{MAJOR}.{MINOR}.{PATCH}` (e.g., `v1.0.0`) |
| Title | `Drawl v{MAJOR}.{MINOR}.{PATCH}` |
| Release notes | Auto-generated by GitHub from merged PRs since previous tag |
| Draft | `false` (published immediately on pipeline success) |
| Pre-release | `false` for tags matching `v*.*.*` exactly |
| Assets | `Drawl.dmg` only |

## Performance Contract

| Metric | Target | Notes |
|--------|--------|-------|
| Total pipeline duration | < 45 minutes | Dominated by Apple notarization (~5–20 min typical) |
| Notarization wait timeout | 30 minutes | `--timeout 30m` in notarytool; job fails if exceeded |
| DMG download size | < 15 MB (app only, no bundled model) | Models are downloaded at first launch |
